# 8.7 UFS PHY Attributes

The MIPI M-PHY includes several configurable attributes. There is range of values defined for the attributes but it is left for the application to fix the actual required values inside the range. Following is the list of such attributes. The UFS application specific requirement for the values can be found from Table 8.1 and Table 8.2.

## Table 8.1 — UFS PHY M-TX Capability Attributes(0)

| Attribute Name | Attribute ID | Range defined in [MIPI-M-PHY] | UFS Value | Notes |
|---|---|---|---|---|
| TX_HSGEAR_Capability | 0x02 | HS_G1_ONLY = 1, HS_G1_TO_G2 = 2, HS_G1_TO_G3 = 3, HS_G1_TO_G3 = 4, HS_G1_TO_G5 = 5 | 5 | 1 |
| TX_PWMGEAR_Capability | 0x04 | PWM_G1_ONLY = 1 | 1 | 2 |
| TX_Amplitude_Capability | 0x05 | SA=1, LA=2, Both=3 | ≥2 | 3 |
| TX_ExternalSYNC_Capability | 0x06 | 0=False 1=True | 0, 1 | 4 |
| TX_HS_Unterminated_LINE_Drive_Capability | 0x07 | 0=No, 1=Yes | 0 | |
| TX_LS_Terminated_LINE_Drive_Capability | 0x08 | 0=No, 1=Yes | 1 | 5 |
| TX_REF_CLOCK_SHARED_Capability | 0x0C | 0=No, 1=Yes | 1 | |
| TX_HibernTime_Capability | 0x0F | 1 to 128 | 1 | 6 |

**NOTE 0** Attributes with the same values as in [MIPI-M-PHY] are not listed in this table. Refer to [MIPI-M-PHY] for other attributes not listed in this table.

**NOTE 1** All HS gears from HS-GEAR1 to TX_HSGEAR_Capability shall be supported.

**NOTE 2** Only PWM-GEAR1 shall be supported.

**NOTE 3** UFS device shall support large amplitude. Support for small amplitude is optional.

**NOTE 4** Support for external SYNC pattern is optional and its definition is device specific.

**NOTE 5** A UFS device shall support terminated LS burst.

**NOTE 6** Time defined in 0.1msec steps.
