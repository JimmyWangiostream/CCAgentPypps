## 8.7 UFS PHY Attributes (cont'd)

### Table 8.2 — UFS PHY M-RX Capability Attributes⁽ᶦ⁾

| Attribute Name | Attribute ID | Range defined in [MIPI-M-PHY] | UFS Value | Notes |
|----------------|--------------|--------------------------------|-----------|--------|
| RX_HSGEAR_Capability | 0x82 | HS_G1_ONLY = 1, HS_G1_TO_G2 = 2, HS_G1_TO_G3 = 3, HS_G1_TO_G4 = 4, HS_G1_TO_G5 = 5 | 5 | 1 |
| RX_PWMGEAR_Capability | 0x84 | PWM_G1_ONLY = 1, PWM_G1_TO_G2 = 2, PWM_G1_TO_G3 = 3, PWM_G1_TO_G4 = 4, PWM_G1_TO_G5 = 5, PWM_G1_TO_G6 = 6, PWM_G1_TO_G7 = 7 | 1 | 2 |
| RX_Hibern8Time_Capability | 0x92 | 1 to 128 | 1 | 3 |
| RX_HS_Equalizer_Setting_Capability | 0x9D | B[0]=0: No equalization B[0]=1: Yes equalization | 1 | |
| RX_EYEMON_Capability | 0xF1 | FALSE=0, TRUE=1 | TRUE=1 | 4 |

**NOTE 0** Attributes with the same values as in [MIPI-M-PHY] are not listed in this table. Refer to [MIPI-M-PHY] for other attributes not listed in this table.

**NOTE 1** All HS gears from HS-GEAR1 to RX_HSGEAR_Capability shall be supported.

**NOTE 2** Only PWM-GEAR1 shall be supported.

**NOTE 3** Time defined in 0.1msec steps.

**NOTE 4** Specifies support for eye monitor function. UFS device and UFS host shall support RX_EYEMON_capability. The rest of the associated attributes for EYEMON shall be taken from MPHY specification.
